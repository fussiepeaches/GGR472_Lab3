## Lab 3: Tokyo Map for Tourists
### Content:
Website that displays the locations of tourist sites in Tokyo, Japan.
### Repo includes:
index.html: HTML file that structures the webpage and renders the map upon opening webpage
style.css: CSS file for styling the webpage and various map elements
script-expressions.js: JavaScript file containing map expressions and events. this inlcudes custom map centred on Tokyo, a geocoder limited to Japan, category classification of various points on the map using different colour schemes, changing point marker opacity depending on if the user is hovering over it, pop-up windows that tell you the name and category of the point when clicked on, and more.
japan_map.geojson: GeoJSON file containing the tourist spot point data with coordinates, names, and categories
